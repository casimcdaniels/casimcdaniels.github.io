Creating an organized react.js project structure

August 04, 2016

If you haven’t heard of ReactJS, it’s an amazing way to build frontend web applications. The library, created by some great people over at facebook, is built upon the View component within the MVC architecture.

Anyways, you wanna get started with building applications in ReactJS, but you’re having trouble making a clean workspace? No worries, I’ll help you out.

To begin. Let’s create a project directory and enter it.

mkdir project
cd project

We want our project to be organized in a more reactive way. so let’s go ahead and think about the structure needed. We want to separate our root directory into three primary folders, components, pages, and assets.

project
  components/
    - app.js
  assets/
    - css/
    - js/
  index.html
